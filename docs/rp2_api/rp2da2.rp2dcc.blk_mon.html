rp2da2.rp2dcc.blk_mon

module documentation

DCC Block Current Detection

This module contains the functions and classes for DCC block detection based on current load monitoring. This is for the resistor based detector where the ADC is on the track side of the galvanic isolation and uses a TI ADC1015 ADC rather than the Pico's own ADC.
